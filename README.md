# Español #

## 1. Instrucciones ##

Gracias a este paquete PHP, podrás cargar cualquier XML de Hordes, Die2Nite o Zombinoia y manejarlo mediante Objetos en PHP. Para preparar los objetos, basta con agregar las siguientes lineas en tu archivo:

>`require_once("swizak.php");`

>`$pagina = new ZomXML("es");`

>`$pagina->load($userkey);`

Si tienes una Key de Desarrollador, simplemente agrega el parámetro al load:

>`$pagina->load($userkey,$devkey);`
